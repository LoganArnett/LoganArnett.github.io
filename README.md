# Logan Arnett — Portfolio

Personal portfolio built with [Qwik](https://qwik.dev/) and [Tailwind CSS](https://tailwindcss.com/), deployed to GitHub Pages via the Qwik static adapter.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Static output is generated in `dist/`.

## Deployment

Automated via GitHub Actions — pushes to `master` trigger a build and deploy to GitHub Pages.
